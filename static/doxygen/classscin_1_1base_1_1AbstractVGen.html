Detailed Description

Represents a template for a parameterizable shader program suitable for composition in a VGen graph.

This class parses incoming VGen definitions and prepares them for efficient processing for shader program generation.

Parameters
nameThe name to use for this VGen, must be unique.
isSamplerIf true indicates this is a sampling VGen
inputsA list of input names, can be empty.
ouptutsA list of output names, must be at least one.
inputDimensionsEach subarray describes the allowable dimensions of the input at that index, and so should have the same number of entries as the number of inputs to the VGen. There should be the same number of subarrays in inputDimensions as there are in outputDimensions, and the values at each index of both arrays form a pair of input/output dimensions. In the event the VGen has no inputs there should be one empty subarray for each output dimension entry.
outputDimensionsDescribes the dimensionality of each output in the input/output dimension pairs formed with inputDimesion.
shaderThe template shader code.
